## Button Styling

### Desktop
- **Variant:** Outline (white background, border)
- **Size:** Default (standard button size)
- **Icon:** Download icon (left side)
- **Text:** "Export CSV"
- **Dropdown:** Chevron down implied by dropdown behavior

### Mobile
- Button adapts to screen size
- May wrap to new line on very small screens
- Dropdown menu adjusts to screen width

## Interaction Flow

1. **User clicks "Export CSV" button**
   - Dropdown menu appears

2. **User selects export option**
   - "Export All Data" â†’ Exports complete dataset
   - "Export Filtered Data" â†’ Exports filtered/sorted data

3. **File downloads automatically**
   - Filename: `{entity}_{type}_{date}.csv`
   - Format: UTF-8 CSV with BOM
   - Opens in Excel/Google Sheets

4. **No page reload**
   - User stays on current page
   - Can continue working immediately

| Page | Location | Filename Pattern |
